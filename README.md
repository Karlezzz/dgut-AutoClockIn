# dgut每日疫情打卡

#### 功能：

选择身体状况，输入体温，提交打卡，判断当前打卡状态，邮件反馈

#### 使用：

1.安装egde及其对应版本的egdedriver

2.如果需要邮件反馈功能，需要邮箱开启IMAP/SMTP服务

3.代码需要完善个人信息
